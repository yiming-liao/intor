# Intor Config

This document lists all available configuration options for **Intor**.

## Configuration Options

#### General

- `id`

#### Locale & Messages

- `defaultLocale` _(required)_
- `supportedLocales` _(required when loader is configured)_
- `fallbackLocales`
- `messages`

#### Translator

- `translator`

#### Routing & Persistence

- `routing`
- `cookie`

#### Message Loading

- `loader`
- `server.loader`
- `client.loader`

#### Observability & Infrastructure

- `logger`
- `cache`
